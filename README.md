# Percolation
A system is modeled by an n x n grid of closed cells.
<br><br>Cells can be: 
* Closed
* Open (empty)
* Open (full)

A cell is said to be *full* if it shares a connection to the topmost row. The system percolates once a *full* cell reaches the bottom row.
